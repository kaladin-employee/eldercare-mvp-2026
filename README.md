# ElderCare Station\n\n## Overview\nThe ElderCare Station is a user-friendly interface designed to be simple, clear, and accessible.\n\n## Features\n- **Home Screen**: Easy-to-read clock and settings button.\n- **Settings Route**: Customizable font size, voice speed, reminders.\n- **Consistent Styling**: Apple HIG/Ive principles applied for simplicity, clarity, and harmony.\n- **Accessibility**: High contrast and large touch targets.
