# ElderCareApp

## Overview
This is a local web app skeleton for the ElderCareApp. It includes basic HTML, CSS, and JavaScript.

## Running
To run this locally:
1. `cd` into the `apps/web` directory.
2. Run `python3 -m http.server 5173 --bind 127.0.0.1`
3. Open your browser to `http://127.0.0.1:5173/`
